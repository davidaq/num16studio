定义：

“16号楼工作室”是一个开发小组。

“16号楼公益站”是一个由“16号楼工作室”开发维护的自助式“公益小站”申请系统。

需求分析：

“16号楼公益站”面向的用户为有意组织公益项目或想为公益事业做宣传的个人或组织。
此人群有兴趣创建/维护一个公益宣传网站，但是缺乏网站制作方面的专业技术，“16号楼公益站”为这人群提供有偿技术支持。

“16号楼公益站”为用户提供简单方便的方式创建、维护一个自己的公益网站，系统为用户提供一套自助建站系统，提供限量的网站功能与傻瓜式的网站后台。

一个公益网站需要拥有的基本功能：
1. 公益项目介绍
2. 公益项目新闻
3. 宣传图画、视频展示
4. 组织机构、个人介绍

扩展功能需求：
1. 接受捐赠
2. 留言
3. 统计图表生成
4. 订阅新闻
5. 投票
6. 图片幻灯
7. 视频播放

【总体而言就是个更专注于某些功能的简化博客系统】

运营方式：

任何人可以凭邮箱注册到一个公益小站，一个邮箱只可以注册到一个小站，每个小站刚注册时可以拥有3天的免费试用时间允许用户体验、学习，之后的使用必须要缴费，欠费的小站会保留7天，7天后未续费一切数据删除。

基本收费方式：
1. 1元/天
2. 15元/月
3. 120元/年

礼券设定：
用户每消费1元钱可获得1个礼券，或者可以直接将1元换为10个礼券。
礼券可用来兑换：
1. 小站主题（2套免费）
2. 扩展功能
3. 可用存储空间（10M免费）
